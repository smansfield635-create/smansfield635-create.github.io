Open only the law you need

Authority lawRoot ownership stays explicit before design, runtime, or presentation claims.

Every chamber needs a route, parent, construct class, and owner function before it can make a larger claim. Compass orients, Door admits, Home stabilizes, Manor maps, Laws governs, Scientific Law tests reality claims, Showroom displays, and Gauges measures.

Route lawEvery chamber must know where it lives and where it sends the visitor next.

No public chamber should float without a parent route, a return route, and a verification path. Laws lives at /laws/. Scientific Law lives under /laws/scientific-law/ as a branch, not a competing parent.

Proof lawGauges, receipts, diagnostics, and visible routes outrank unsupported claims.

Diagnostics stay visible before ornament. A visual page can look better without proving readiness. The page must keep visual polish, source truth, scientific testability, and generation readiness separate.

Runtime lawMotion must stay contained, readable, route-safe, and owned by the correct chamber.

Static route chambers must not load heavy render chains. Animation may express a route field, but runtime, renderer, instruments, and diagnostics must not leak into rooms that only need links.

Generation lawBaseline, readiness, and generation labels must not blur into false completion.

A baseline is an accepted state, not a claim that all future generations are complete. Marker health, visual quality, scientific confidence, and generation readiness remain distinct.

Product and planet separation lawProducts stay products. Planetary constructs stay planetary constructs.

ARCHCOIN, AI, Five Flags, Education, and Nutrition are product chambers under Products. Manor, Laws, Scientific Law, Universe, Frontier, and world surfaces remain estate, law, or planetary constructs.

Human boundary lawSupport, explanation, preparation, and proof must not become reckless authority.

Medical, legal, financial, safety, scientific, and official claims must mark what still needs qualified professional or source verification. Assistance organizes; it does not impersonate authority.

Interaction lawUsers must be able to tell what can be touched and what cannot.

Interactive diamonds, buttons, and links must be visually distinct. Decorative centers and noninteractive marks must not pretend to be primary click targets.

Receipt lawEvery major renewal must leave a receipt.

The receipt must name what changed, what stayed untouched, what route owns the page, what dependencies were not mutated, and whether a visual pass is claimed.

Visual claim lawDo not overclaim what a page only appears to show.

Generated images, graphic boxes, screenshots, animation, and polish do not equal source truth. If visual approval has not been explicitly accepted, visual_pass_claimed remains false.

Foreground lawLaws, Entry, Proof, and Return are foreground controls.

The foreground core holds the safety loop: Laws governs, Entry admits, Proof measures, and Return re-anchors. Scientific Law branches from Laws and tests reality claims without replacing the governing chamber.

Access lawLaw must remain reachable from the public route system.

Compass, Door, Home, Manor, Laws, Scientific Law, Showroom, Products, Frontier, Universe, and Gauges must remain reachable so the law layer does not become isolated.

Stale language lawRetired labels must not keep controlling current pages.

Old Demo Universe, Earth G1, Planet 1, flat/round/globe mode, or other retired page identities must not remain as active control language unless a route intentionally preserves them.

Animation lawMotion expresses hierarchy; it does not replace hierarchy.

Figure-eight traversal and diamond motion may show relationship, but the page must still be readable without motion and must respect reduced-motion preferences.

Scope lawFull-file TNT only for touched files.

A file change must be delivered as a full replacement for the touched file. No patches, fragments, or partial diffs control the project state.

Return lawEvery outward route needs a way back.

Expansion remains coherent only when return routes stay visible. Scientific Law must return to Laws, and Laws must return to Compass, Door, Home, Manor, and Gauges.
