Choose the next path without losing the thread.

Explore is the public route-map surface. It separates destinations by purpose so the system stays clear.

Compass gives the command surface. Door creates entry. Home stabilizes. Explore shows the available paths: book, universe, products, proof, diagnostics, laws, frontier, and return routes.

A map is not more noise. A good map removes noise.
The visitor should never lose the thread just because the world got bigger.
Explore keeps the system from becoming a hallway with too many doors.
