TRAJECTORY

Before constraint-first specs, behavior drifted into ad-hoc code and ambiguous terms.

SSG compresses behavior into five primitives and gates advancement by receipts.

Constraint-native specs become replay-safe execution contracts across domains.

PrimitivesR/S/C/G/Guards

  • Stores: where state lives
  • Constraints: invariants
  • Goals: success conditions
  • Guards: admissibility checks
