Execution Space

A simple model of how decisions become real over time.

What you’re seeing

This is a map of possible paths, not a command. It shows how a system can move across time using four basic actions.

  • S = Selection (what enters)
  • R = Resolution (how constraints apply)
  • C = Commitment (what becomes real)
  • N = Continuation (what feeds back)
Why “64”

If you look three steps deep and at each step you can choose one of four actions, there are:

4 × 4 × 4 = 64 possible sequences.

That’s what “bounded state space (64)” means: the complete set of three-step patterns in this four-action model.
