The big idea

The discovery is architectural before it is commercial.

Energy loop navigator

The loop decides whether the system works.

Step 1 Storage

Where is potential held safely? Practical endpoints require stored potential before use.

Step 2 Release

How is stored potential converted into controlled work? Release must be governed before the operating regime begins.

Step 3 Operate

What high-energy regime or useful process happens? Plasma is an operating regime, not fuel or storage.

Step 4 Recover

What value, heat, charge, material, or information is retained? Open loops burn. Closed loops endure.

Step 5 Storage Return

Can the system return to a stable usable state? The loop must return value after input, losses, and reset costs.

Loop condition Useful output must survive the full cost. Useful Output ≥ Total Input + Losses + Reset Costs.
CLEAS navigator

CLEAS is the grammar of lawful energy.

CLEAS means Closed-Loop Energy Architecture Standard. It is not a reactor, plasma device, consumer product, power seller, or deployment claim. It is the standard that tells the system where energy belongs, how risk is separated, and what must be measured before claims scale.

CLEASPlacement

High-energy regimes belong upstream where the failure envelope can be governed.

CLEASMeasurement

Input, output, reset cost, degradation, and stable operating window must be measurable before claims scale.

CLEASFailure Envelope

Risk must be contained before it reaches human proximity, consumer endpoints, or ungoverned infrastructure.

CLEASEndpoint Eligibility

Endpoints receive passive stored potential only, not plasma or fusion operating regimes.

CLEASSafety Boundary

Energy density, operating risk, containment, and auditability define the safe deployment boundary.

CLEASLicensing Path

Commercialization begins with standards, data, partner validation, and licensing before direct deployment claims.

PSALM gates

Five gates keep Energy lawful.

PPlacement

High-energy regime upstream. Stored potential downstream. No collapse of upstream danger into endpoint proximity.

SStorage

No stored potential, no practical endpoint use. Plasma is not storage. Fusion is not endpoint storage.

AAdmissibility

Before deployment, coordination must be stable enough for measurement, governance, repair, and scaling.

LLoop

Storage → Release → Operate → Recover → Storage. Open loops burn. Closed loops endure.

MMobility

Fusion may power the system that charges the vehicle. Fusion does not belong inside the vehicle.

Compressed law Risk upstream. Use downstream. High-energy regimes stay in hardened infrastructure. Endpoints use passive stored potential.

Engineering notes

Technical language for deeper readers.

Engineering appendix Open technical depth.
Energy input accounting Inputs must be named. Solar, grid, thermal, chemical, battery, or other support inputs must be tracked before useful-output claims scale.
Reset cost The next run matters. A system that cannot reset without losing the value it created has not closed the loop.
Thermal loss Heat is not automatically useful. Useful output depends on what can be retained, converted, stored, or safely rejected.
Material degradation Operating regimes consume hardware. Plasma and fusion-adjacent regimes must account for degradation, maintenance, and replacement cost.
Storage carrier eligibility Endpoint use needs stable carriers. Batteries, fuels, thermal stores, mechanical stores, or grid-connected storage carry practical downstream use.
Failure envelope Risk must stay contained. High-energy operating regimes require hardened placement, auditability, isolation, and fail-safe behavior.
Grid / microgrid placement Infrastructure absorbs complexity. Fusion may eventually support the system that charges or fuels endpoints, but it does not belong inside endpoints.
Loop closure criteria The loop must prove retained value. Storage, release, operation, recovery, reset, and return must remain measurable across repeated load.
Kill conditions

The contract stops if the claim outruns the evidence.

Safety stop Open the failure conditions.
  • Stop if plasma is treated as storage.
  • Stop if fusion moves to the endpoint.
  • Stop if the loop cannot be defined.
  • Stop if reset cost exceeds useful output.
  • Stop if degradation exceeds tolerance.
  • Stop if the failure envelope reaches human proximity.
  • Stop if measurement cannot be audited.
  • Stop if deployment depends on consumer exposure.
  • Stop if commercial claims exceed measured evidence.
  • Stop if the narrative turns energy science into magic.
