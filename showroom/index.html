Open the construction identity

The G3 Diamond Lattice is a purpose-built computational object.

It is not an imported model, a generated image, a video, or a static illustration.

Its visible surface and internal lattice originate from one deterministic geometry authority.

Open the geometry record

The model is governed by 16 radial sectors and 16 structural bands.

Their intersection establishes 256 addressable structural seats.

The exterior contains 512 rendered triangles and the internal field contains 800 lattice connections.

Open the material record

The Diamond uses 14 material regions across the table, crown, bridge girdle, pavilion, culet, lattice, and active inspection state.

The visible architecture is organized around a gold crown, a cyan-white illuminated bridge girdle, a sapphire pavilion, and a gold-cyan internal lattice.

Open the rendering record

The object is expressed through a native WebGL renderer without an external 3D rendering library.

The renderer owns shaders, buffers, lighting, rotation, zoom, facet selection, context-loss handling, and the projected ground shadow.

Open the interaction record

The public interface exposes two accepted views: Object View and Lattice View.

The object supports pointer rotation, touch rotation, pinch zoom, wheel zoom, idle rotation, reset, inspection, reduced motion, responsive resizing, and WebGL fallback.

Open the ownership boundary

The HTML owns the route shell, exhibit architecture, accessible controls, narrative presentation, dialog, and fallback placement.

The geometry file owns the model. The renderer owns visual expression. The UI controller owns the accepted public object controls.

The floating Map / Portal blip remains the sole public entryway into Mirrorland.
