Admissibility Preconditions

Lens is set on Tier 1 and applies here.
1 · Coherence Precedence
External coherence cannot exceed internal self-coherence.
You can’t reliably project integrity you don’t actually have inside.
2 · Admissibility
A system either admits execution or it does not.
If execution isn’t structurally allowed, effort doesn’t matter.
3 · If-and-Only-If Execution
Outcomes occur only when all required constraints are simultaneously satisfied.
Almost doesn’t execute. Missing one required condition breaks the result.
