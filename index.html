Methods

Geodiametrics classifies systems from evidence traces. It excludes narrative and intent inference. It enforces provability through explicit constraints and auditable artifacts.

0–13 Structural Cycle

Observer-invariant lifecycle classifier. Evidence-only. No intent inference.

Scope: public artifacts with measurable ingress and identifiable structure. Rule: classify from evidence traces; exclude narrative/intent.

Findings

Findings are stated in bounded form. Each finding is evidence-addressable and audit-friendly.

Scientific Findings

S1 — Provability from structure
Execution is valid only when constraints are satisfied and outcomes are traceable through artifacts.

S2 — Receipts block silent change
A transition exists only when it emits a receipt. Absence of receipt invalidates advancement.

S3 — Drift is observable
Drift is detected by violated constraints, bypassed gates, or mixed-layer operations.

S4 — Trajectory is admissibility-bounded
Constraint tightening changes reachable futures without changing evidence rules.

S5 — Lossless compression requires invariants
Compression is valid only when invariants preserve meaning and reproducibility.

Categorical Findings

C1 — Execution band (0–7)
Formation→constraint→boundary→executable minimum→stability→projection→feedback→completion.

C2 — Visibility band (9–13)
Visibility→closure→extension→higher-order execution→re-stabilization.

C3 — Closure rule
Closure: additions fail without explicit exceptions. Extension: invariants hold but are insufficient.

Independent Findings

I1 — Reproducible classification
Independent observers classify the same state from the same trace under the same rules.

I2 — Verifiable outcomes
Outcomes verify by receipt chain and log inclusion without narrator trust.

I3 — Objective drift detection
Drift is detected from constraint violations and missing gates.

I4 — Audit without authority
Audit relies on artifacts, not identity.

Method Summary

Core loop: CHECK → RECEIPT → LOG → SEARCH → PROOF. A result exists only when it is traceable.

Boundary rule: internal meaning and external causality remain separated. Mixed-domain claims are rejected.
