Measurement rules

What Gauges is allowed to claim.

SHOWROOM_ROUTE

Passes when the route serves.

Showroom route status passes only when `/showroom/` and `/showroom/globe/` serve live content.

TRANSITION_LAYER

Passes when the new contract is visible.

The transition layer passes only when our-universe reference, Demo Planet destination, seven-landmass language, and receipt markers are source-visible.

INSTRUMENT_CONTRACT

Passes only when the expected API exists.

The instrument contract passes only when the asset exposes the expected Showroom globe object and render function. Otherwise the state is HOLD or FAIL.

UNIVERSE_ROUTE

Passes when the map route exists.

`/nine-summits-universe/` must exist and carry the world-map receipt before Showroom can rely on it.

OWNER_CONFIRMATION

Gauges verifies source state.

Gauges does not replace owner visual review. If the live page shows fallback, the next repair must target the owning route or instrument.

NO_DRIFT

Page roles stay separate.

Showroom inspects, Nine Summits Universe explains, and Gauges measures. This page must not become the Showroom or the Universe page.
