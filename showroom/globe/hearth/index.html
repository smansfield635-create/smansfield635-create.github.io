About Hearth
Hearth gives survival a visible condition.

Hearth is not a decorative planet page. It is the survival-path field where the visible planet, inspection controls, completion latch, and route diagnostics must survive together.

The page now obeys shell-first architecture: HTML paints public structure first and avoids blocking the browser with runtime orchestration.

Runtime Cycle
The HTML file does not conduct the runtime.

HTML loads only the East entry after shell stability. East then stages downstream runtime discovery without allowing HTML to serially load Runtime Table, source stack, Canvas, West, North, or South.

Cycle 1 returns proof to North. Cycle 2 reaches Canvas only after North, East, South, and West complete the lawful release path.
