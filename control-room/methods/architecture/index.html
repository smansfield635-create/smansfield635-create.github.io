Architecture

Structural map of the system. Defines authority boundaries, discovery rules, and execution geometry.

Authority Layers

  • Control Room — sole execution authority.
  • Engine Room — execution workspace.
  • Engine Matrix — read-only registry.
  • Dashboard — observer-only summary.
  • Apps — viewer/launcher.

Discovery & Registry

  • Prefix Law: discover engines via ENGINE::.
  • Device-Local: state lives in localStorage.
  • Deterministic: discovery by prefix scan only.

Execution Geometry

  • Quad-Core Fusion: Selection → Resolution → Commitment → Continuation.
  • Namespaces: stable identifiers per core.
  • Isolation: no implicit cross-core mutation.
