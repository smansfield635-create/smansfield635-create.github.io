Methods

Constraint-first diagnostic system. Evidence-only. No intent inference. This page defines invariant laws, categorical structure, and audit limits.

Core Loop

CHECK → RECEIPT → LOG → SEARCH → PROOF
Results exist only when traceable.

Scientific Findings

S1 — Provability from structure. Execution admissible only when constraints are satisfied and outcomes trace.
S2 — Receipts block silent change. No receipt, no state transition.
S3 — Drift is observable. Detected via violated constraints or bypassed gates.
S4 — Trajectory is bounded. Tightening constraints changes reachable futures.
S5 — Invariants preserve compression. Loss of invariants produces false positives.

Categorical Findings

C1 — Execution band (0–7). Formation → constraint → boundary → executable minimum → stability → projection → feedback → completion.
C2 — Visibility band (9–13). Visibility → closure → extension → higher-order execution → re-stabilization.
C3 — Closure rule. Additions fail unless explicit exceptions are introduced.

Independent Findings

I1 — Reproducible classification. Independent observers converge on the same state.
I2 — Verifiable outcomes. Verified via receipts and append-only logs.
I3 — Objective drift detection. No narrative inference required.
I4 — Audit without authority. Validity governed by artifacts, not identity.

Audit Limits

No inference of intent, motive, personality, or internal cognition. Mixed-domain claims are rejected. Missing receipts invalidate results.
