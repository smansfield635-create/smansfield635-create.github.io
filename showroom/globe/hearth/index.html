About Hearth
Hearth gives survival a visible condition.

Hearth is not a decorative planet page. It is the survival-path field where visible body, inspection controls, completion latch, and route diagnostics must survive together.

This page is now the receiving shell. It does not serial-load the heavy stack. It waits for the selector and allows release only through the selector API.

Runtime Cycle
The HTML file does not conduct the runtime.

The HTML shell exposes stable anchors and receives the female JS selector. The selector decides whether the shell is matched before any runtime queue is allowed.

Runtime release is intentionally manual in this shell to prevent a flash-to-white-screen collapse while the paired system is being stabilized.
