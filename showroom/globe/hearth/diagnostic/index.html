North · Decision

North is the decision layer. It reads the receipt stream and determines the next lawful move. It should point to one next file and one next action. North does not become Canvas authority.

Return to Orbit

East · Intake

East is the intake layer. It verifies that diagnostic child scripts and route receivers are loaded. If an authority is missing, East tells the chamber whether this is a load, cache, script-order, or alias issue.

Return to Orbit

West · Boundary

West reads the rendered target, endpoint family, LabWest separation, and Canvas Boundary. If an authority is found but not callable, the problem is a method surface. If it is not found, the problem is script loading, cache, or alias publication.

Return to Orbit

South · Output

South organizes proof output, sidecar readings, receipt packets, and copy/export controls. South keeps the raw proof stream intact while the readable gauges explain what happened.

Return to Orbit

Gauges

Gauges are readable interpretations of receipt state. Found means an authority exists. Callable means it exposes a direct method. Missing means the live resolver could not find it. Protected means the chamber confirms no mutation authority.

Return to Orbit

Receipts

Receipts are machine-readable packets. They preserve fields such as RUN_STATE, TRUST_STATE, BLOCKING, NEXT_FILE, and NEXT_ACTION. The UI can be modern, but the raw receipt packet stays copyable and stable.

Return to Orbit

Protected Systems

This chamber does not authorize production mutation, Canvas build, Canvas release, controls mutation, or runtime restart. Any receipt generated here must keep those fields false.

Glossary

  • Authority — a public object such as HEARTH.diagnosticWest.
  • Alias — a public path used to locate an authority.
  • Manifest — the diagnostic child script loader.
  • Direct method — a callable method such as runDiagnostic() or getReport().
  • Next Lawful Move — the chamber’s recommended next file and next action.
  • Boundary — a no-touch protection line around production systems.
